# Workspace Skills

`bootstrap/SKILLS.md` is the local skill catalog for this workspace.

Use it this way:
- Read it after `PRODUCT.md`, `AGENTS.md`, and `bootstrap/USER.md`.
- Use it to decide which `bootstrap/skills/<skill-name>/` directory is relevant.
- Keep actual instructions and helper assets inside each skill directory, not in this index file.
- Update this file when a skill is added, removed, or repurposed.

Current skills:
- `workspace-bootstrap`: orientation for the shared workspace contract.
